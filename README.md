# Task Manager App

A full-stack Task Manager application built with:

## Setup and Installation

### 1. Clone the repository

git clone https://github.com/vakkko/Vladimer_Gabisonia_helfy_task.git
cd task-manager

## Running the Backend

The backend runs on:

http://localhost:4000

Start the backend:

cd backend
node server

Note: Tasks are stored in memory. Restarting the server will reset all tasks.

---

## Running the Frontend

cd frontend
npm install
npm run dev

The frontend will run on:

http://localhost:5173

Make sure the backend is running on port 4000 before using the frontend.
